Overview

2026 年上半年，Agent × Skill 成为 LLM agent 研究的一条高度收敛主线：以 Anthropic Claude Skills 为代表的"可复用能力模块"被大量工作重新定义并验证，研究问题从"agent 能不能用 skill"转向"skill 如何被 自动产生、跨任务迁移、大规模管理"。本综述覆盖 Hugging Face Papers 上 2026-02 至 2026-04 间的 28 篇相关论文。

我们按 skill 生命周期 组织全文：方向一聚焦技能如何 产生（合成 / 自进化 / 沉淀）；方向二聚焦技能如何 累积并复用（记忆 / 失败复用 / 元学习）；方向三聚焦当技能库规模膨胀后如何 管理、检索与评测（基建）。带 精读 标记的论文提供独立详情页，含方法图解、完整实验表与讨论笔记。

01 · Synthesis, Self-Evolution & Consolidation

核心问题：如何让 agent 从轨迹中自动 产生 可复用 skill，并通过迭代精炼持续进化。代表性范式有三条——轨迹到声明式技能的蒸馏（Trace2Skill），RL 驱动的层次化技能发现（SkillRL），以及"强模型沉淀 → 弱模型即插即用"（SkillX）。

03 · Infrastructure — Management, Retrieval & Benchmarks
